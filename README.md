## Работа с CLI

### Достаточно ввести команду
```
python -m src.main
```

### После этого вам будут доступны 6 алгоритмов сортировок(bubble_sort, radix_sort, quick_sort, heap_sort, bucket_sort, count_sort).
Вводите имя из списка выше после комманды sorts, после этого вводите через пробел числа массива для сортировки, если вы используете bucket_sort, то можно указать праметр  --buckets.
Пример (если есть отрицательные числа, то перед их вводом нужен знак --)
```
sort quick_sort -- 2 5 7 0 -1 3 0 2
sort heap_sort 2 5 7 0 1 3 0 2
```

### Также вам доступны две структуры: стек и очередь. После каждого запуска они очищаются
```
queue dequeue 24
```
